This folder contains Jupyter Notebook examples demonstrating how to use the package and reproduce results.

    DataExtraction.ipynb, DataVisual.ipynb, and DataPreprocessing.ipynb
        - These notebooks outline the steps for extracting, cleaning, and preprocessing data from the JSMS databook, converting it into well-structured 	Excel files.

    Data_Que.ipynb
        - This notebook explores data-related questions, such as whether increasing data volume improves results, with a specific focus on aluminium.

    DoE1, DoE2, DoE3, DoE4, and DoE5
        - These notebooks document different experimental approaches for predicting fatigue parameters in aluminium and steel.
